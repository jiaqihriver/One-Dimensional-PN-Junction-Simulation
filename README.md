# 一维理想 PN 结稳态仿真

本项目实现了一个基于耗尽近似和 Shockley 模型的一维理想 PN 结稳态仿真工具。通过调节掺杂浓度和外加偏压，可实时计算并展示能带图、电势分布、电场分布、载流子浓度分布以及 I-V 特性曲线。

## 主要功能

- **动态参数调节**：通过滑块实时改变 P 区掺杂浓度 \(N_a\)、N 区掺杂浓度 \(N_d\) 和外加偏压 \(V_a\)。
- **核心物理量计算**：内建电势 \(V_{bi}\)、耗尽层宽度 \(W\)、最大电场强度等。
- **多维度可视化**：
  - 能带图（导带 Ec、价带 Ev、本征能级 Ei、准费米能级 Efp/Efn）
  - 静电势分布
  - 电场分布
  - 载流子浓度分布（对数坐标）
  - 理想二极管 I-V 特性曲线及当前工作点

## 物理模型与假设

- **材料**：硅（Si），温度 \(T = 300\,\text{K}\)
- **模型**：突变结耗尽近似
- **载流子统计**：玻尔兹曼近似（非简并）
- **电流模型**：理想 Shockley 方程
- **准费米能级**：在耗尽区内考虑电子和空穴准费米能级分离

## 使用方法

1. 打开 `index.html` 文件（所有代码已整合在单个 HTML 中）。
2. 左侧控制面板调节参数：
   - **P 区掺杂浓度 \(N_a\)**：范围 \(10^{13} \sim 10^{18}\,\text{cm}^{-3}\)
   - **N 区掺杂浓度 \(N_d\)**：范围 \(10^{13} \sim 10^{18}\,\text{cm}^{-3}\)
   - **外加偏压 \(V_a\)**：范围 \(-2.0 \sim 0.85\,\text{V}\)
3. 右侧图表将自动更新，展示当前偏压下的器件内部特性。
4. 鼠标悬停在图表上可查看详细数值。

## 技术栈

- HTML5 / CSS3

- [Plotly.js](https://plotly.com/javascript/) 绘图库

## 文件说明

- `index.html`：包含完整的仿真界面、样式、脚本和物理计算逻辑，无需额外依赖即可运行。

## 注意事项

- 当正向偏压接近或超过内建电势时，模型会自动钳位，防止耗尽层宽度变为负值。
- 载流子浓度采用对数坐标，便于观察耗尽区内的指数变化。
- I-V 曲线显示理想二极管特性，饱和电流为预设参考值，实际物理器件需根据扩散长度修正。
